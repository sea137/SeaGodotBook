# 🎮 SeaGodotBook —— 从零开始的 Godot 4 游戏开发教程

欢迎！这是一套**开源、免费、持续更新**的 Godot 4 教程，适合编程初学者和独立游戏开发者。

## 📚 如何使用本教程

- 左侧 **侧边栏** 按章节顺序阅读
- 右上角 **搜索框** 可快速定位知识点
- 代码示例均经过测试，可直接复制使用

## 🚀 你将学到

- GDScript 语法精讲
- 2D/3D 场景构建
- 信号系统与 UI 制作
- 发布到 Windows / Web / 移动端
