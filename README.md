# SeaGodotBook —— 从零开始的 Godot 4 游戏开发教程

<img src="https://godotengine.org/themes/godotengine/assets/logo.svg" width="120" align="right">

欢迎来到 **SeaGodotBook**！这是一套**开源、免费、持续更新**的 Godot 4 教程，适合编程初学者和希望转型独立游戏开发的你。

## 📖 如何使用本教程

- 左侧 **目录** 按章节顺序阅读
- 右上角 **搜索框** 可快速定位知识点
- 所有代码示例均可 **一键复制**

## 🚀 你将学到

- GDScript 语法精讲
- 2D/3D 场景构建
- 信号系统与 UI 制作
- 发布到 Windows / Web / 手机
